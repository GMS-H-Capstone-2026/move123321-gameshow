# RN 실습 체크리스트

이 폴더에서 오늘 작업합니다.

## 먼저 볼 문서

React Native 문법이 낯설면 먼저 읽습니다.

```txt
../docs/RN-BASICS.md
```

## 실행

```bash
npm install
npm run web
```

이미 설치했다면 다음부터는 이것만 실행합니다.

```bash
npm run web
```

## 오늘 수정할 파일

```txt
App.js
components/GameCard.js
data/games.js
```

## 에이전트 사용 방식

에이전트에게 완성본을 한 번에 맡기지 않습니다.

- 먼저 내가 보고 있는 파일과 하려는 작업을 말합니다.
- 수정할 위치와 순서를 먼저 확인합니다.
- 작은 단위로 고친 뒤 바로 실행해서 확인합니다.
- 에러가 나면 에러 메시지와 내가 바꾼 코드를 함께 보여줍니다.
- 에이전트의 답을 붙여넣기 전에 어떤 줄이 바뀌는지 읽습니다.

## 과제 진행 순서

### 1. 화면 먼저 확인

- 브라우저에 게임 카드가 보이는지 확인합니다.
- 검색창에 `네온`을 입력해봅니다.
- `RPG`, `액션`, `인디` 버튼을 눌러봅니다.
- 게임 카드를 눌러 상세 모달을 엽니다.
- 장바구니 담기 버튼을 눌러 숫자가 올라가는지 봅니다.

### 2. 게임 데이터 추가

파일:

```txt
data/games.js
```

작업:

- 기존 게임 하나를 복사합니다.
- `id`는 겹치지 않게 바꿉니다.
- `title`, `category`, `genres`, `rating`, `price`, `image`, `description`을 바꿉니다.
- 화면에서 새 게임이 보이는지 확인합니다.

확인:

- 데이터 형식이 기존 게임들과 같은가?
- 새 게임의 `category` 값이 카테고리 버튼과 연결되는가?

### 3. 카드 디자인 수정

파일:

```txt
components/GameCard.js
```

작업:

- `styles.card`의 배경색을 바꿉니다.
- `styles.title`의 글자 크기나 색상을 바꿉니다.
- 버튼 문구나 버튼 색상을 바꿉니다.
- 화면에서 카드가 깨지지 않는지 확인합니다.

확인:

- 제목과 가격이 겹치지 않는가?
- 버튼을 눌렀을 때 기존 기능이 유지되는가?

### 4. 검색 범위 넓히기

파일:

```txt
App.js
```

찾을 코드:

```js
const filteredGames = useMemo(() => {
```

작업:

- 현재 검색은 제목과 장르를 확인합니다.
- `description`도 검색되도록 조건을 추가합니다.
- 설명에 있는 단어로 검색해서 결과가 나오는지 확인합니다.

확인:

- 빈 검색어일 때 전체 게임이 보이는가?
- 제목, 장르, 설명 검색이 모두 되는가?

### 5. 새 카테고리 추가

파일:

```txt
App.js
data/games.js
```

작업:

- `App.js`의 `categories` 배열에 새 카테고리 버튼을 추가합니다.
- `data/games.js`에서 게임 하나의 `category` 값을 새 카테고리와 같게 바꿉니다.
- 새 카테고리 버튼을 눌러 필터가 되는지 확인합니다.

확인:

- 버튼의 `value`와 게임의 `category` 값이 같은가?
- `전체` 버튼을 누르면 다시 모든 게임이 보이는가?

### 6. 상세 모달 정보 추가

파일:

```txt
App.js
data/games.js
```

작업:

- `data/games.js`에 새 정보 하나를 추가합니다. 예: `platforms`, `difficulty`, `releaseYear`
- `App.js`의 `Modal` 안에 그 정보를 보여줍니다.
- 게임 카드를 눌러 모달에서 확인합니다.

확인:

- 모든 게임 데이터에 같은 속성이 있는가?
- 정보가 없을 때 화면이 깨지지 않는가?

### 7. 나만의 기능 하나 추가

아래 중 하나를 고릅니다.

- 쉬움: 카드에 `추천` 뱃지 추가
- 쉬움: 할인 전 가격 함께 보여주기
- 중간: 평점순 정렬 버튼 추가
- 중간: 찜하기 버튼 추가
- 어려움: 장바구니 목록 보여주기
- 어려움: 구매 완료 버튼 만들기

확인:

- 새 기능이 실제 화면에서 동작하는가?
- 어떤 파일을 왜 바꿨는지 설명할 수 있는가?
